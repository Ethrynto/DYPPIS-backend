# Media files and images

### Get image by ID
`[GET] /api/v1/media-storage/{id}` - Get image by ID <br>

### Add the new image
`[POST] /api/v1/media-storage"` - Create image. Requirements:
- `file: file`
- `category_id: string`

### Delete the image by ID
`[DELETE] /api/v1/media-storage/{id}` - Delete image by ID <br>
